# Installation

### Prerequisites

- C++ compiler

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/Jastreas/CPP_Small_Programs_Collection.git
   ```
2. Open any of the programs:
    **Example:**
    cd CPP-BankingBalance
3. Compile the code:
    **Example:**
    cpp main.cpp -o main
4. Run the program:
    **Example:**
    ./main.exe (this works on windows)
